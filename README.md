# water-quality-portfolio
# 💧 Water Quality Analysis & Dashboard

## 📌 Overview
This project analyzes water pollution using Python and provides an interactive dashboard.

## 🚀 Features
- Trend analysis (COD, BOD)
- Correlation analysis
- Interactive dashboard (Streamlit)
- Environmental insights

## 📊 Preview
![Preview](./assets/preview.png)

## ⚙️ Tech Stack
- Python
- Pandas
- Matplotlib
- Streamlit

## ▶️ How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run analysis
cd scripts
python analysis.py

### 3. Run dashboard
cd ../dashboard
streamlit run app.py

## 📈 Key Insight
- Location A3 shows highest pollution
- Strong correlation COD-BOD
- Increasing trend over time
